# Demos e industrias — GetSocial Theme v1.4

## Plantillas de inicio + preset de color

Cada demo tiene **dos pasos** para verse coherente:

| Demo | Plantilla inicio | Preset de color (Configuración del tema → Colores) |
|------|------------------|-----------------------------------------------------|
| Moda | `index` (predeterminada) | **Moda** o preset tema «Moda (Fashion)» |
| Belleza | `beauty` | **Belleza** |
| Electrónica | `electronics` | **Electrónica** |
| Hogar | `home` | **Hogar** |
| Gourmet | `food` | **Gourmet / Food** |
| Deporte | `sport` | **Deporte** |

### Activar plantilla de inicio

1. **Personalizar tema** → selector de página → **Inicio**
2. ⚙️ de la página → **Cambiar plantilla** → elige `beauty`, `electronics`, etc.

### Activar colores

**Opción A — Selector rápido**

1. **Configuración del tema** (engranaje) → **Colores**
2. **Preset de color (por industria)** → elige la industria
3. Guardar

**Opción B — Preset completo del tema**

1. En la biblioteca de temas, al instalar o duplicar, Shopify puede mostrar presets: *Moda*, *Belleza*, *Electrónica*, etc.
2. Cada uno aplica colores + radios de borde acordes

**Opción C — Personalizado**

- Preset **Personalizado** usa los selectores de color manuales de abajo.

## Cajón de filtros en móvil (v1.4)

En **colecciones**:

- **Móvil**: botón **Filtrar** abre panel lateral; **Ver resultados** aplica y cierra (con AJAX activo).
- **Escritorio**: filtros fijos en columna izquierda (mismo formulario).

Ajustes: sección **Colección** → *Cajón de filtros en móvil* / *Filtros AJAX*.

Requiere app **Search & Discovery** para definir filtros.
