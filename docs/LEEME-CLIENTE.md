## Instalación

1. En Shopify: **Tienda online → Temas → Añadir tema → Subir archivo ZIP**
2. Sube **este mismo ZIP** o, si tienes ambos archivos, usa `getsocial-theme-v1.6.0.zip` (solo tema, sin documentación extra).
3. Activa el tema y abre **Personalizar**.

> Si ves el error *missing layout/theme.liquid*, descarga de nuevo el ZIP desde el enlace oficial (versiones antiguas generadas en Windows podían usar rutas incorrectas).

### Con Shopify CLI

```bash
cd carpeta-del-tema
shopify theme dev --store TU-TIENDA.myshopify.com
```
